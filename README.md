# TrustNet

Distributed blockchain infrastructure installation and setup.

## Quick Start

```bash
curl -fsSL https://raw.githubusercontent.com/jcgarcia/TrustNet/main/install.sh | bash
```

## What is TrustNet?

TrustNet is a distributed consensus network built with Tendermint.

## Installation

The installer will:
- Create QEMU virtual machines (trustnet-node, trustnet-registry)
- Install Alpine Linux 3.22.2 ARM64
- Configure IPv6 ULA networking
- Deploy Tendermint consensus node
- Setup container registry
